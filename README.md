## Fitur
- PID control (Kp, Ki, Kd)
- Analog feedback (potensiometer)
- Arah motor via DIR pin
- Output PWM

## Cara Install Arduino IDE
1. Download ZIP dari GitHub.
2. Arduino IDE → Sketch → Include Library → Add .ZIP Library.

## Cara Install Platform IO
1. Buka platform.ini
2. tambahkan lib_deps=https://github.com/bransazza/gibranpid

## Contoh
Ada di folder examples/BasicPID.
